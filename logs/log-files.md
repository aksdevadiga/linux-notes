# Linux Log Files

Logs are records of system and application events.
They are essential for monitoring, debugging, and troubleshooting issues
in Linux systems.

Most log files are stored in the `/var/log` directory.

---

## /var/log Directory

Contains system and application log files.

**Example:**
ls /var/log

Common log files include:
- syslog
- auth.log
- kern.log
- boot.log

---
