# Common Linux Commands

## pwd
Prints the current working directory.

**Syntax:**
pwd

**Example:**
pwd

**Explanation:**
Shows the full path of the directory you are currently in.

---

## ls
Lists directory contents.

**Syntax:**
ls [options] [path]

**Example:**
ls -l /var/log

**Explanation:**
Displays files with permissions, owner, size, and date.
