# File Permissions in Linux

## chmod
Changes file permissions.

**Numeric Example:**
chmod 755 script.sh

**Symbolic Example:**
chmod u+x script.sh

**Explanation:**
Controls read, write, and execute permissions.

---

## chown
Changes file owner.

**Example:**
chown user:user file.txt

---

## umask
Sets default permission mode.

**Example:**
umask 022
